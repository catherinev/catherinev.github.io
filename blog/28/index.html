Using LyX to Display Code: Part 1

Prerequisites
Note: the introduction to this tutorial can be found here.

0. Mac OS X

As I mentioned in the intro, I have a Mac, so this tutorial will be specifically aimed at Mac OS X. However, it may not be too hard to modify it for Linux (and possibly Windows), especially if you are familiar with LaTeX and how it is configured on your machine.

1. LaTeX

You need to have LaTeX install on your computer. I strongly recommend downloading MacTeX, although other distributions are available. The package we will be using, minted, requires several packages that are included MacTeX, as well as the editor TeXShop. These are not all included in more basic distributions of TeX and could make the rest of the installation process much more challenging.

If you already have TeXLive 2014 but are using a different text editor (not TeXShop), that's also fine.

2. LyX

LyX can be found at this website: http://www.lyx.org/

3. Python

minted requires Python. If you just downloaded LyX, you should be in good shape (it ships with a recent version of Python). However, if you are using an old version of LyX, make sure you are running at least Python 2.6. If you need to update, check out this site: https://www.python.org/

4. Pygments

minted also requires Pygments, which can be installed with the following command:

sudo easy_install Pygments

If you already have Pygments, make sure you have at least version 1.2.
