Using LyX to Display Code: Part 1

Note: the introduction to this tutorial can be found here.

Prerequisites

1. LaTeX

You need to have LaTeX install on your computer. I strongly recommend downloading TeXLive or MacTeX. The package we will be using, minted, requires several packages that are included in both of these distributions of LaTeX. However, they are not all included in more basic versions and could make the rest of the installation process much more challenging.

2. LyX

LyX can be found at this website: http://www.lyx.org/

3. Python

minted requires Python. If you just downloaded LyX, you should be in good shape (it ships with a recent version of Python). However, if you are using an old version of LyX, make sure you are running at least Python 2.6. If you need to update, check out this site: https://www.python.org/"

4. Pygments

minted also requires Pygments, which can be installed with the following command:

sudo easy_install Pygments

If you already have Pygments, make sure you have at least version 1.2.
